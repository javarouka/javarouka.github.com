쿠키 4 사전 과제

각기 다른 팀에서 작업한 JavaScript 파일 3개가 있다.

brands.js, restaurant.js, main.js 이 파일 셋이며 당신은 main.js 를 담당하는 팀이다. 이제 main.js 의 나머지를 채워서 나머지 작업을 마무리해야 한다.

그런데 파일을 보니 당장 사용하기엔 문제가 있어 보인다. 변수와 함수 이름이 중복되어, 충돌하고 있기 때문이다.
어떻게든 brands.js, restaurant.js를 잘 수정하고, main.js 를 채워 예제를 완성시켜보자.

Point 화면이 이렇게 (Click) 나오면 가산점.

Point brands.js, restaurant.js의 변수 및 함수 이름을 수정하지 않고 동작하게 만들면 가산점.
Point 자신이 수정한 이유에 대해 충분한 주석으로 설명이 있다면 가산점.


TIP! 변수의 유효범위 및 Module 화에 대해 생각해보자

질문은 메일로 :) 

/* @file brands.js */
var data = [
    '레드윙',
    '호킨스',
    '닥터마틴',
    '클락스',
    '버팔로',
    '팀버랜드'
];

function render($parent) {
    var $brandList = $("<ul></ul>");
    $.each(data, function(index, item) {
        $brandList.append("<li>" + item + "</li>")
    });
    $parent.append($brandList);
}
/* @file restaurant.js */
