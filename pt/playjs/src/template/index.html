Object

얼마나 알고 있나요?

JavaScript 의 Object 는 Key-Value Pair.

즉, Dictionary 나 Map 으로 불리는 것과 비슷하다.

게다가 JS 객체는 Object 와 모두 연결되어 있다


const good = {};
good.job = '잘했어';
good.day = () => new Date();
               

const one = { /* some implements */ };

alert(one) // "하나!"
alert(99 + one) // 100
JSON.stringify(one) // "변환되었습니다"
