CSS 선택자 샘플입니다

Type Selector
div p a {
	font-size: 120%;
	color: green;
	font-weight: bold;
}

a 링크의 기본 동작은 윈도우의 URL을 href 속성값으로 교체하는 것입니다.

anchor
ID Selector
#paragraph1, #paragraph3 {
	background-color: #666;
	color: #eee;
}

p 엘리먼트는 문장을 표현합니다.

블럭 엘리먼트이며, 인라인 요소를 가질 수 있습니다.

대부분의 브라우저에서 p 태그의 기본 스타일은 줄 바꿈과 일정 간격을 띄어놓는 형태입니다.
