Creating your first chart visually

Here, we will get our hands down n dirty and create our first chart using the FusionCharts XT XML Generator Tool. As our first example, we will create a simple Monthly Sales Chart.

When we will be finished, our chart will look as under:

Pre-requisites for creating a chart

To create any FusionCharts XT chart, you need to assemble the following four things:

  1. SWF File for the chart that you want to create. We want to create a 3D Column Chart, whose SWF file is named as Column3D.swf. All the chart SWF files are present in Download Package > Charts folder.
  2. FusionCharts JavaScript Files - These files (FusionCharts.js, FusionCharts.HC.js, FusionCharts.HC.Charts.js, and jquery.min.js) are present in Download Package > Charts folder.
  3. XML Data File. As you already know by now, FusionCharts XT accepts data only in a pre-defined XML format. We will build the XML data using our visual tool soon.
  4. HTML Wrapper File. This HTML file will contain the code to embed the chart. Again, we will build this using the visual tool.

Setting the playground

We now need to set the playground for our first chart. Create a new folder named as MyFirstChart under any folder on your computer (say c:\FusionCharts, if you can't think of a better place). For each chart that you create using the FusionCharts XT visual GUI, you will need to create a separate folder.

You can also have multiple charts in same folder. But, for that you will need to do some code editing which is discussed in "Guide for Web Developers" section. Here, we have tried to keep things as simple as possible.

Next, you need to copy the chart SWF file from Download Package > Charts and the JavaScript files (FusionCharts.js, FusionCharts.HC.js, FusionCharts.HC.Charts.js and jquery.min.js) from Download Package > Charts folder respectively into this folder. The setup should look as under till now:

That completes our setup for the chart creation. In this folder, we can now have one chart. For each chart that you create, you will need to create a separate folder.

With the playground ready, it is now just a two-step affair to pop-up enticing charts:

  1. Generate XML
  2. Create HTML Wrapper

Let's now shift our attention to creating the XML data file for this chart. XML generation is again split into following:

  1. Actual Data for plotting the Chart
  2. Chart Cosmetics like Caption, sub-caption, Titles for X and Y-axis and so on.

XML for Chart Cosmetics

Let's now add some captions and axis names to our chart. To do so, click the "Configure Chart" button. You will be provided with an interface to enter titles and additional parameters as under:

As you can see above, you can provide the caption, sub-caption, x-axis name, and y-axis name for the chart. You can also enter a number prefix for the chart. Number prefix refers to a character which gets added to all the numbers on the chart. Since in our case, we are displaying sales figures in $, we set $ as number prefix. Similar to number prefix, number suffix characters gets added at the end of all the numbers on the chart.

Next, FusionCharts XT allows you to choose a palette (color theme) from five pre-defined palettes. Lastly, you can choose whether to format your numbers by scaling them into K (Thousands), M (Millions). If set to yes, 13000 will appear as 13K on chart, 12340000 will appear as 1.234M, and so on.

FusionCharts XT offers a lot more configurable properties, which can be configured via XML only. Those properties cannot be configured using this GUI.

Enter the above parameters and click the "Update" button. You will now see that the XML has been updated with this information:

You now need to save this XML data for use by the chart. To do so, click the "Copy to Clipboard" button. This copies the XML data to your clipboard. Next, paste it in a text editor and save it as Data.xml in our previous MyFirstChart folder.

Now, the final step left is to create the HTML code for the chart, which we will see next.

Creating the HTML Wrapper for chart

To create the HTML code for this chart, click the "Generate HTML" button in the above screen. You will be presented with the following screen where you can select your chart type, width and height (in pixels). Select Column 3D as chart type and enter your desired width and height as under:

You will see that when you change the chart type or dimensions, the HTML code in the text-area below gets dynamically updated. You now need to save this HTML code. Again use the "Copy to Clipboard" button and save this HTML as Chart.html in the same folder.

And that marks the end of chart creation process. To view the chart, explore to our MyFirstChart folder and run Chart.html in your browser. You will get an animated Column 3D chart as shown below.
