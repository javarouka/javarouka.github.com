JSON

JSON 변환을 Native로 지원

ECMA Script 3nd

(function(){
  var jsonString = "{ \"hello\": \"world\" }";

  // eval 사용
  alert((eval("("+jsonString+")")));

  // 동적 함수 사용
  alert((new Function("return " + jsonString))());
})();
실행

ECMA Script 5th

(function(){
  var jsonString = "{ \"hello\": \"world\" }";

  // Native JSON Object 사용
  alert(JSON.parse(jsonString));

  // 추가적으로 JSON 객체의 serialize 도 지원
  alert(JSON.stringify(JSON.parse(jsonString)));

})();
실행
